# Keyboard Shortcuts

The following shortcuts are available throughout MyApp. On macOS, replace Ctrl with Cmd.

## App

| Shortcut | Action |
|---|---|
| Ctrl+/ | Show keyboard shortcuts |
| Ctrl+H | Search and Replace (in editor) |

## Editor

These shortcuts work when the editor has focus.

| Shortcut | Action |
|---|---|
| Ctrl+B | Bold |
| Ctrl+I | Italic |
| Ctrl+U | Underline |
| Ctrl+Shift+X | Strikethrough |
| Ctrl+E | Inline code |
| Ctrl+Shift+1 | Heading 1 |
| Ctrl+Shift+2 | Heading 2 |
| Ctrl+Shift+3 | Heading 3 |
| Ctrl+Shift+8 | Bullet list |
| Ctrl+Shift+9 | Ordered list |
| Ctrl+Shift+B | Blockquote |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
